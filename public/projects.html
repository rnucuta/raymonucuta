Changelog
A lot has happened over the past years. Here's some of the highlights:
  • January 2019
    On track to finish Edubase
    EduBase is an educational app for developing countries built for KaiOS, a cheap, low-powered operating system commonly found on "feature phones". Check out the project: EduBase on Github.
  • January 2019
    Finished UPenn Intro to Robotics Specialization
    I have been working on UPenn Intro to Robotics Specialization for over a year. I learned about aerial robotics, MATLAB, Raspberry Pi, optics, and much more! Just finished it!
  • January 2019
    Finished ResearchKit
    Chrome and Firefox extension that helps doing research. Just highlight something, right click it, and select ResearchKit. Four windows will pop up displaying summaries of various encyclopedia sites. You can find the source code on GitHub: ResearchKit. You can also download it on Chrome and Firefox here.
  • December 2018
    Finished YT Barebones
    After experimenting with my Raspberry Pi Zero, I discovered that I couldn't view YouTube on it. So, I made a website that uses Google's V3 API's to display YouTube in basic HTML. Check it out here: YT Barebones. You can find the source code on Github.
  • November 2018
    Finished Ram Munchies
    Fun website that crashes your web browser if you use Windows or crashes your computer if you use a Unix based OS. It also has some cool animations and stores cookies: Ram Munchies. You can find the source code on Github.
  • November 2018
    Finished prototype for Engine Testing Website
    Finished prototype for an aircraft engine testing website during my partnership with Kronos Consulting, LLC. Built with SQL, Python, Javascript, HTML, CSS, and Visual Basic.
  • April 2017
    Initial Deploy of WikiQuery
    Website built on Ruby on Rails that searches terms on Wikipedia and provides summaries. Does so with Wikipedia's API's. Check it out here. The source code is on Github.
  • October 2016
    Finished Gas power motorized bike project
    I was working on my gas power motorized bike for a while, but it ended up dying on me after 3 rides after a chain seize. I've spent many hours going down a hill and popping the clutch hoping it would start. Unfortunately, there hasn't been any luck for the past few years, and I still have no idea why it is broken. Arizona sun?
